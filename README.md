## Planar Atlas - A Magic: The Gathering Planechase Tool
![Planar Atlas](/images/assets/social-preview.jpg)

A feature complete Planechase tool for Magic: The Gathering! Browse a searchable gallery of every plane and phenomenon including custom sets, all tagged and sorted with multiple views. Build, store, and share multiple decks, and simulate entire Planechase games in both classic shared deck and Blind Eternities Map modes.

### Features
- **Gallery** — Searchable, filterable card database with multiple views and lots of options.
- **Deck Builder** — Build, name, and save multiple deck slots with import/export support.
- **Planechase Simulator** — Classic mode and Blind Eternities Map, onboarding, and even savestates using seeds.
- **Custom Cards** — Includes community-created planes alongside official content, curated for the best experience.
- **Profile Seeds** — Export and import your entire setup (preferences + all decks) as a short seed string rather than an account system.
