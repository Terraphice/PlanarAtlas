# Planechase Booster — Terraphice
Planechase Booster — Terraphice is a custom set for Magic: The Gathering's Planechase format, focused on delivering a balanced, enjoyable to play, and flavorful experience. The set consists of existing Plane and Phenomenon card types, and generally seeks not to introduce new mechanics not already existing in MTG, somewhere, unlike some other custom Planechase design.
(Exceptions made for a small subset of cards with very thoughtful design and flavor.)

This repository is composed of the set-files for Magic Set Editor used to develop the cards, for ease of use, editing, updating, and learning. It also includes full-quality images for each of the cards in the set, exported by MSE, for ease of use and hosting purposes.
